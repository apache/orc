Mailing Lists

There are several development mailing lists for ORC:

  • dev@orc.apache.org - Development discussions with archive here
  • issues@orc.apache.org - Bug tracking with archive here
  • commits@orc.apache.org - Git tracking with archive here

You can subscribe to the lists by sending email to list-subscribe@orc.apache.org and unsubscribe by sending email to list-unsubscribe@orc.apache.org.

Source code

ORC uses git for version control. Get the source code:

% git clone https://git-wip-us.apache.org/repos/asf/orc.git

The important branches are:

  • master - The trunk for all developement
  • asf-site - The pages that are deployed to https://orc.apache.org/

Please check our coding guidelines.

Reviews

All code must be +1’ed by a committer other than the author prior to its commit.
